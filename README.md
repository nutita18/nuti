# Idle School Tycoon: โรงเรียนเล็ก ฝันใหญ่

เกมบริหารโรงเรียนแบบ Idle / Tycoon สำหรับเล่นบนมือถือและคอมพิวเตอร์

## ระบบในต้นแบบเผยแพร่
- ระบบงบประมาณ คะแนนความรู้ และชื่อเสียง
- อาคาร ห้องเรียน โรงอาหาร ห้องสมุด ห้องคอม สนามกีฬา ห้องวิทยาศาสตร์
- ครูประจำอาคารพร้อมสกิลเร่งรายได้
- ภารกิจ ความสำเร็จ และการเลื่อนระดับโรงเรียน
- รายได้ออฟไลน์และบันทึกเกมในอุปกรณ์

เกมพัฒนาเป็น Web App/PWA พร้อมนำไปเผยแพร่ผ่าน GitHub Pages
